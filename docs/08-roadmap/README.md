# Roadmap

The honest sequence for building Tacet — [your home on the open social web](../07-brand/README.md).

## How to read this

Two things are true at once, and we keep them straight:

- **The principles are fixed.** People before posts. Relationships before engagement. Identity before platforms. Calm before addiction. Open before closed. These do not move between versions. See [founding principles](../00-manifesto/founding-principles.md).
- **The sequence is honest.** We do not describe unbuilt things in the present tense. This roadmap says plainly what exists today, what must be reshaped, and what must be built.

Governing rule, on every page: **code follows philosophy.** The order we build
in is chosen to protect the principles, not to chase features.

## The arc

- **[MVP](mvp.md)** — one owned identity, your people, Today, and the first honest window into the Fediverse: read, follow, reply across the open web.
- **[v1](v1.md)** — full Compose, Communities, Conversations, Discover; a calm home a person prefers over the seven apps it replaces.
- **[Future](future.md)** — deeper federation, portability you can feel, mobile, calm defaults — becoming the best home for the open social web.

## The re-founding (important)

Tacet is being re-founded around `docs/`. There is a working, invite-gated app
today on Cloudflare Workers + D1 + R2, with accounts, profiles, following,
communities, posts, replies, and reactions. But it was built under an **old
thesis** — a "quiet invite-only clubhouse / rooms" idea — and has **not** yet
been reshaped to the direction in these docs. See `../../PRODUCT_DIRECTION.md`
for the legacy assumptions being overwritten.

Crucially: **federation is not built yet.** The current app is local-only. Every
mention of the Fediverse in this roadmap is work ahead of us, described honestly.

So this roadmap is partly *build* and partly *reshape*. Each page below marks
which is which.
