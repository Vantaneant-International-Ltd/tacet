# Visual System V2

> This milestone assumes **Product UX Pass** is complete.
>
> Do **NOT** redesign workflows.
>
> Do **NOT** add features.
>
> Do **NOT** change information architecture.
>
> Do **NOT** change navigation.
>
> Do **NOT** introduce new product concepts.
>
> This milestone exists for one purpose:
>
> **Make Tacet feel like the product it was always meant to be.**

---

# Philosophy

This is not a redesign.

This is refinement.

Everything already works.

The architecture is complete.

The product is coherent.

Now make every interaction feel intentional.

Imagine Tacet being introduced on stage at WWDC.

It should feel:

- inevitable
- calm
- timeless
- premium
- human

Never trendy.

Never flashy.

Never "startup."

---

# Inspiration

Quality bar only.

Do NOT imitate.

Inspired by:

- Apple
- Linear
- Arc Browser
- Notion
- Things
- Ivory
- Bear
- Craft

Not inspired by:

- Twitter/X
- Threads
- Instagram
- LinkedIn
- Discord
- Slack
- Crypto products
- Generic SaaS dashboards

The goal is to create something recognisably **Tacet**.

---

# Mission

Create the definitive visual identity for:

> **Your home on the open social web.**

Every screen should communicate calm confidence.

Nothing should feel accidental.

---

# Principles

People before posts.

Relationships before engagement.

Identity before platforms.

Calm before addiction.

Open before closed.

Every pixel should reinforce these principles.

---

# Absolutely no feature work

Do not add:

- buttons
- menus
- cards
- tabs
- pages
- metrics
- interactions
- animations that introduce functionality

Only refine what already exists.

---

# Typography

Review every heading.

Review every body style.

Review spacing.

Review line height.

Review hierarchy.

Reduce visual noise.

Increase readability.

Typography should carry the interface.

Not borders.

---

# Layout

Review:

- spacing
- margins
- padding
- rhythm
- alignment
- proportions

Everything should feel balanced.

Nothing should feel cramped.

Nothing should feel oversized.

---

# Components

Review every component.

Buttons.

Cards.

Inputs.

Navigation.

Composer.

Profile.

Collections.

Saved.

Badges.

Search.

Dialogs.

Sheets.

Menus.

Everything.

Make them feel like they belong to one design system.

---

# Navigation

Keep navigation exactly as defined during Product UX Pass.

Improve only:

- spacing
- weight
- typography
- interaction feedback
- transitions

---

# Cards

Review every card.

Should they breathe more?

Need less chrome?

Better hierarchy?

Better emphasis?

Simplify.

---

# Motion

Motion should communicate meaning.

Not decoration.

Examples:

- page transitions
- hover
- focus
- loading
- save
- open
- dismiss

Motion should feel closer to Apple than social media.

Slow enough to feel deliberate.

Fast enough to never block.

---

# Colour

Refine the palette.

Improve contrast.

Improve accessibility.

Avoid oversaturation.

The interface should disappear behind the content.

---

# Icons

Review every icon.

One coherent icon language.

Consistent stroke.

Consistent sizing.

Consistent spacing.

---

# Empty states

Every empty state should feel welcoming.

Not broken.

Guide gently.

Never overwhelm.

---

# Loading

Loading should feel calm.

Use skeletons where appropriate.

Avoid unnecessary spinners.

Avoid flashing layouts.

---

# Responsive

Perfect:

Desktop.

Tablet.

Mobile.

Large displays.

Ultra-wide.

Everything should feel intentionally designed.

Never merely scaled.

---

# Accessibility

Review:

Contrast.

Keyboard navigation.

Screen readers.

Focus states.

Touch targets.

Reduced motion.

Font scaling.

Accessibility is part of the product.

Not an afterthought.

---

# Dark mode

Dark mode should not simply invert colours.

It should feel intentionally designed.

Depth should come from contrast.

Not from black backgrounds.

---

# Light mode

Light mode should feel warm.

Quiet.

Readable.

Never sterile.

---

# Images

Review media presentation.

Cropping.

Corner radius.

Spacing.

Loading.

Captions.

Everything should feel editorial.

---

# Microinteractions

Review:

Saving.

Pinning.

Collections.

Notes.

Opening profiles.

Navigation.

Search.

Every action should acknowledge the user.

Subtle.

Never playful.

Never distracting.

---

# Brand

Strengthen Tacet's identity.

Without shouting.

The product should become recognisable from:

Typography.

Spacing.

Motion.

Hierarchy.

Not logos.

---

# Performance

Beauty must never compromise speed.

Animations should never make the interface feel slower.

Every frame matters.

---

# Remove before adding

Whenever tempted to add decoration:

Remove something instead.

Less is usually better.

---

# Final review

Before considering this milestone complete, ask:

Would Apple ship this?

Would Linear accept this?

Would Arc Browser accept this?

Would this still feel modern in five years?

Most importantly:

Does this feel unmistakably like Tacet?

If the answer is no,

continue refining.
