# Tacet landing UI kit

Marketing surface — the only place the full dark hero treatment (deep ink + iris glow + blossom corner + breathing constellation) is used.

- `index.html` — hero with top nav, breathing mark, headline, two CTAs, manifesto strip.
- Static HTML/CSS on tokens; no bundle dependency.
- Rules: one glowing primary CTA per page; constellation opacity ≤ 0.55; ambient motion uses `--ease-breathe` and respects `prefers-reduced-motion`.
